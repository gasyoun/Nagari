EmEditor Devanagari Transliteration Encoding Converters (21 VBA Scripts for Sanskrit)

Convert huge devanagari documents in seconds. No batch function, one document at a time.

- https://groups.google.com/forum/#!forum/nagari - support. http://is.gd/PpHcVr thread.
